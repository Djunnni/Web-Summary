### CSS display 속성

블록

-   항상 새로운 줄에서 시작하며, 너비를 다 차지한다.

h1, p, form, ul, li

인라인

-   새로운 줄에서 시작하지 않으며, HTML 요소의 내용만큼만 차지

span a img

display속성은 자유롭게 변경은 가능하다. 인라인 요소 내부에 다른 요소를 추가할 수는 없다.
애초에 display속성값이 block인 경우에만 다른 요소를 포함할 수 있다.

인라인 블록

-   인라인 요소처럼 동작한다. 하지만 요소 내부에서는 block 요소처럼 동작한다.
-   너비와 높이 지정 가능
-   margin을 통해 여백 지정가능
